# ChatCord
<p> Chatcord webapp is a platform where you can chat and discuss your problems related to coding and programming</p>


# Prerequisites
<ul>
  <li>Nodejs</li>
  <li>MongoDB</li>
  <li>Socket.io</li>
  <li>Bootstrap4</li>
  </ul>
  

# Installing
<p> Download or Clone the Project.</p>
